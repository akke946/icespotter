# ICE-Spotter
A german web-app developed in HTML to **keep track of spotted trains**. Supports custom databases, sighting history and customizable themes.
Now supports other operators!
## Usage
You can use the web-app by visiting akke946.github.io/icespotter! *Spot and settings data saves to your browser's localStorage.*
*You may also download the app for local usage*, the most recent release is included as a .zip file under **Releases**
## Supported operators
- DB Fernverkehr AG for their ICE trains
- Hallesche Verkehrs-AG (HAVAG) for their trams
